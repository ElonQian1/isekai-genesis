# 多人卡牌游戏

## 项目结构
- `packages/client`: 前端 (React + Vite)
- `packages/server`: 后端 (Node.js + Express + Socket.IO + SQLite)
- `packages/shared`: 共享类型

## 快速开始

1. **安装依赖**
   在根目录下运行:
   ```bash
   npm install
   ```

2. **启动开发环境**
   在根目录下运行:
   ```bash
   npm run dev:server
   ```
   打开另一个终端运行:
   ```bash
   npm run dev:client
   ```

3. **访问**
   打开浏览器访问 http://localhost:5173
